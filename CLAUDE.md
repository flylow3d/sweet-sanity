# Sweet Sanity Ice Cream — concept website (spec build #2)

Concept/spec website for **Sweet Sanity Ice Cream ("and More")**, 15512 Lima
Rd, Huntertown, IN 46748 · (260) 580-3475. Seasonal hometown scoop shop in a
renovated garage. Built by It's Already Done Web Co. from public information —
**the owners have not seen or approved it**; footer carries the concept
disclaimer. Portfolio piece on alreadydoneweb.com.

  Hawaiian shave ice; 2 oat-based non-dairy flavors; 30+ toppings; four sizes

  CONFLICTING sources exist; site says "seasonal, check Facebook" everywhere

  Drummer); new owners since ~2023 expanded the menu; shares building with
  Stella's Alterations; indoor + umbrella patio seating; cards/contactless OK

## Design

Playful parlor: cream bg, strawberry pink primary, mint green secondary,
chocolate-brown text, cherry accent. Matches their real pink/green/white
decor. Fonts: Baloo 2 (display) + Nunito. Signature motif: CSS sprinkles
divider strip. 3 pages: index / menu (Flavors & Treats) / visit.
